# 通用答题系统

## 系统概述

这是一个基于网页的通用答题系统，使用 Bootstrap 框架构建界面，支持多种题型测试和答题管理，包含前台答题和后台管理功能。

## 主要功能

- ✅ 试卷选择与加载
- ✅ **学员姓名输入**（新增！）
- ✅ 实时答题计时
- ✅ 题目标记功能
- ✅ **答题记录管理**（新增！）
- ✅ **查看所有学员答题详情**（新增！）
- ✅ 后台管理入口（点击页面顶部的"后台管理"按钮）
- ✅ 答题卡导航
- ✅ 自动保存答题进度
- ✅ 试卷提交与评分
- ✅ 后台试卷上传与管理
- ✅ **局域网访问支持**

## 🔥 新增功能

### 1. 学员姓名输入
学员在答题前需要输入姓名，系统会记录每位学员的答题情况。

### 2. 答题记录管理
- 自动保存所有学员的答题记录
- 查看每道题目的学员选择
- 按试卷、按学员姓名筛选记录
- 统计平均分、提交人数等数据

### 3. 答题详情查看
管理员可以在管理页面查看：
- 学员姓名
- 试卷名称
- 提交时间
- 得分情况
- 每道题目的选择答案

## 局域网部署指南

### 快速启动（推荐）

1. **双击 `启动服务器.bat`**
   - 系统会自动检查 Node.js 是否安装
   - 自动安装所需依赖
   - 自动启动服务器
   - 显示本机 IP 地址

2. **其他电脑访问**
   - 在浏览器中输入：`http://服务器IP:3000`
   - 例如：`http://192.168.1.100:3000`

### 手动启动（高级用户）

1. 确保已安装 Node.js（[下载地址](https://nodejs.org/)）

2. 安装依赖：
   ```bash
   npm install
   ```

3. 启动服务器：
   ```bash
   node saveTest.js
   ```

4. 修改服务器端口（如需要）：
   编辑 `saveTest.js`，修改 `const PORT = 3000;`

### 查看本机 IP 地址

在命令提示符中输入：
```bash
ipconfig
```
找到"IPv4 地址"那一行，例如：`192.168.1.100`

## 使用指南

### 学员答题流程

1. 打开答题系统（`index.html` 或通过服务器访问）
2. 从下拉菜单中选择试卷
3. 点击"加载"按钮开始答题
4. **输入学员姓名**（必填）
5. 答题过程中系统会自动保存进度
6. 点击"提交试卷"完成测试
7. 系统会显示得分，并保存答题记录

### 管理员查看答题记录

1. 点击页面右上角的**"后台管理"**按钮
2. 进入答题记录管理页面（`admin.html`）
3. 查看所有学员的答题情况：
   - **学员姓名**
   - **试卷名称**
   - **提交时间**
   - **得分及正确率**
   - **每道题目的选择答案**
4. 使用筛选功能按试卷或学员姓名查找记录
5. 查看统计数据（总人数、总试卷数、平均分）
6. 如需删除记录，点击对应记录右侧的"删除"按钮

### 上传试卷

1. 在管理页面点击"返回答题系统"
2. 点击"后台管理"进入上传页面
3. 填写试卷信息（ID、名称、描述）
4. 选择 JSON 格式的试卷文件
5. 点击"上传试卷"完成上传

## 注意事项

- ⚠️ 启动服务器前请确保 3000 端口未被占用
- ⚠️ 局域网访问时，请确保防火墙允许 3000 端口的入站连接
- ⚠️ 学员姓名必须填写，否则无法提交试卷
- ✅ 系统会自动保存未完成的答题进度
- ✅ 重置答题将清除所有答题记录
- ✅ 上传的试卷文件会自动保存到 tests 目录
- ✅ 答题记录会永久保存在服务器端（answer_records.json）

## 常见问题

### Q: 其他电脑无法访问？
A: 检查以下几点：
1. 服务器是否已启动（查看命令窗口）
2. 防火墙是否允许 3000 端口
3. 确保使用正确的 IP 地址
4. 确保所有电脑在同一局域网内

### Q: 如何修改端口？
A: 编辑 `saveTest.js`，找到 `const PORT = 3000;`，将 3000 改为你想要的端口号。

### Q: 需要安装什么软件？
A: 只需要安装 Node.js（[官网下载](https://nodejs.org/)）

### Q: 答题记录保存在哪里？
A: 答题记录保存在服务器目录下的 `answer_records.json` 文件中，重启服务器不会丢失。

### Q: 如何删除某个学员的答题记录？
A: 在管理页面（admin.html）中，找到对应的记录，点击右侧的"删除"按钮即可。

### Q: 可以按学员姓名查找记录吗？
A: 可以！在管理页面上方的筛选栏中，输入学员姓名，点击"筛选"按钮即可。

## 技术支持

如果遇到问题，请检查：
1. Node.js 是否正确安装（命令行输入 `node --version`）
2. 依赖是否安装成功（检查是否有 node_modules 文件夹）
3. 端口是否被占用（尝试更换端口）
4. 服务器是否正常运行（检查命令窗口）
